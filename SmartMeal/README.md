# smartmeal

SmartMeal, una aplicación móvil multiplataforma desarrollada con Flutter que permite a los usuarios generar menús semanales equilibrados y automatizar su lista de la compra, integrando funciones de reconocimiento inteligente de productos y almacenamiento en la nube
